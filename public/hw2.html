10 Findings:

  1. There are so many ways to embed a video. Before, using the <object> tag was the way to embed videos. But now, using iframe is the more preferred way since it lets the browser determine what player to use instead of checking for a plugin (e.g. Flash)
  2. The usage for the <hr> tag changed between HTML 4 and HTML 5. In HTML 4, it was used as a generic horizonal rule. However, in HTML 5, it is mainly used to represent thematic breaks between paragraphs or sections of a chapter.
  3. Many tags for formatting were deprecated in favor of using style sheets to achieve the same results since style sheets can offer more flexibility and functionality. Most of the "other tags" I found were these types of tags.
  4. oninvalid is not supported on safari
  5. onmousewheel is deprecated
  6. onerror overrides the onsubmit on chrome
  7. onmouseleave and onmouseout look like are consolidated into onmouseout
  8. bdi element is not supported in all browsers
